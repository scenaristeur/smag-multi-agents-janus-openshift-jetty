Smag, est une application un peu particulière, elle met à votre disposition des agents autonomes afin de faciliter la résolution de problèmes complexes. Pour vérifier que ces agents sont disponibles, vous pouvez cliquer sur le lien "Hello Servlet", vous verrez normalement deux agents Albert et Bernardo avec un numéro identifiant. Si c'est le cas, c'est que le système fonctionne correctement.
Smag s'appuie sur un serveur RDF pour représenter les informations (voir le blog).
Pour ajouter des projets, utilisez l'application Android Projet.apk
ou le programme pour Windows (en cours de développement)
